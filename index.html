🏭 Maker Space MVP

Веб-приложение для управления университетским Maker Space

📋 Об приложении

Это комплексное веб-приложение для управления работой университетского Maker Space. Система обеспечивает учёт пользователей, контроль допуска к оборудованию на основе инструктажа, резервирование оборудования и ведение полного аудита действий.

⚠️ Важно: Это демо-версия показывает структуру и возможности проекта. Для полноценного использования требуется локальное развёртывание Django приложения.

✨ Основные возможности

👤 Система ролей

4 типа пользователей: Guest, Student, Staff, Admin с разными уровнями доступа

📅 Резервирование

Бронирование оборудования с шагом 20 минут, макс. 24 часа вперёд на 3 недели

📊 Расписание

Общее расписание загрузки оборудования по дням и часам

🔐 Инструктаж

Контроль доступа к оборудованию на основе пройденного обучения

🔔 Уведомления

Внутренняя система уведомлений по событиям бронирования

📝 Аудит

Полный журнал всех действий пользователей и системы

🔑 Роли и доступ

Роль Описание Что может делать
Guest Гость Просмотр оборудования, бронирование без инструктажа
Student Студент Полное бронирование после инструктажа, просмотр своих броней
Staff Сотрудник Автоподтверждение броней, проверка заявок, управление инструктажем
Admin Администратор Все права Staff + админ-панель Django, управление всеми данными

📋 Правила резервирования

Временные рамки

• Мин. длит.: 20 мин
• Макс. длит.: 24 часа
• Шаг времени: 20 мин
• Макс. вперёд: 3 недели

Процесс подтверждения

• Обычные пользователи: ручное подтверждение
• Staff/Admin: автоматическое подтверждение
• Таймаут без ответа: 7 дней

Ограничения

• Запрещены пересечения броней
• Недоступно на downtime периодах
• Требуется инструктаж для ряда оборудования

🔧 Модули проекта

Users

Регистрация, логин, профиль, роли, управление инструктажем

Equipment

Каталог оборудования, категории, расписание, downtime периоды

Reservations

Создание, подтверждение, отклонение, отмена и продление броней

Notifications

Внутренний центр уведомлений по событиям системы

Audit

Журнал всех действий пользователей и системы

Main

Главная страница и базовые сущности приложения

⚡ Быстрый старт (локально)

# 1. Клонирование
git clone https://github.com/MishOchEK-1/MSproject.git
cd MSproject

# 2. Виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate  # или .venv\Scripts\activate на Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Миграции
python manage.py migrate

# 5. Демо-данные
python manage.py seed_demo_activity --days 12

# 6. Запуск
python manage.py runserver

# 7. Откройте http://127.0.0.1:8000/
