# MSprojectMVP

MVP Django-проекта для сайта цеха Maker Space от лица вуза.

## Что уже заложено

- базовая архитектура проекта на Django;
- кастомная модель пользователя с ролями;
- подготовленные доменные приложения:
  - `users`
  - `equipment`
  - `reservations`
  - `notifications`
  - `audit`
- базовая шаблонная главная страница;
- SQLite на старте;
- базовые тесты и миграции.

## Стек

- Python
- Django
- SQLite

## Быстрый старт

```bash
python manage.py migrate
python manage.py runserver
```

## Статус

Сейчас реализован первый инфраструктурный блок: фундамент проекта, настройки, кастомный пользователь и подготовка к дальнейшей разработке логики бронирования.
